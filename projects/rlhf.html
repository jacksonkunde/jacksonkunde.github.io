Credit: Huggingface

Project Overview

Here, I implement a popular deep algorithm for aligning and training language models called Reinforcement Learning from Human Feedback [RLHF]. This implementation really gets into the details, you should check it out. I demonstrate how we can train a language model to exhibit different behaviors based a reward function.

Results with '.' Reward Function

We define our reward function as the number of periods in the generated text. It seems silly, but this will give us a sense of how our reward function changes our generated text.
